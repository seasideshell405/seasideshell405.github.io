# 为什么选择 Eleventy？

在众多静态网站生成器中，我最终选择了 Eleventy。这篇文章聊聊选择它的理由。

## 零配置起步

Eleventy 的核心理念是"零配置也能用"。只需要一个 Markdown 文件，就能生成一个完整的页面。当你需要更多控制时，再逐步添加配置。

## 灵活的模板系统

Eleventy 支持多种模板语言：Nunjucks、Liquid、Handlebars、Pug、EJS 等，甚至可以混用。这意味着不同的页面可以使用不同的模板引擎。

## 出色的构建性能

Eleventy 的构建速度非常快，即使有上百个页面，构建也只需数秒。这对于开发体验来说至关重要。

## 纯静态输出

不像某些框架会注入大量 JavaScript，Eleventy 默认输出纯 HTML，没有任何多余的运行时负担。这让它非常适合内容型网站。

## 社区生态

虽然 Eleventy 相对年轻，但社区活跃，文档完善。配合 Netlify、Vercel 或 GitHub Pages 可以实现自动化部署。

## 小结

如果你在寻找一个轻量、灵活、快速的静态网站生成器，Eleventy 绝对值得一试。它不是最花哨的工具，但可能是最务实的那个。
